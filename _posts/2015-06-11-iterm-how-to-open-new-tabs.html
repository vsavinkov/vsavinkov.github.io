--- layout: post title: 'iTerm: how to open new tabs programmatically' date: '2015-06-11T10:08:00.001+03:00' author: Daemonic tags: - iTerm - bash - MacOS modified_time: '2015-06-11T10:11:37.356+03:00' blogger_id: tag:blogger.com,1999:blog-2174521382944768960.post-2349952759759495500 blogger_orig_url: http://my-irregularity.blogspot.com/2015/06/iterm-how-to-open-new-tabs.html --- This script will open new tab in iTerm and execute command there.
Useful when you need to do something in several tabs (deploy applications, pull repos, etc.).

tab() {
  osascript \
    -e 'activate application "iTerm"' \
    -e 'tell application "System Events" to keystroke "t" using command down' \
    -e "tell application \"iTerm\" to tell session -1 of current terminal to write text \"$1\""
}

Can be modified to use regular Mac Terminal instead of iTerm - just change app name in 3rd string.

Examples:

tab 'cd ~/Projects; git pull'
tab 'ls'
